## 🌟 About

This is my personal developer portfolio — a single-page website that presents who I am, what I build, and how to reach me. No frameworks, no dependencies. Just clean, semantic HTML with custom CSS styling and vanilla JavaScript for interactivity.

**Live:** [View Portfolio](https://waseefullahh.github.io/portfolio) *(deploy via GitHub Pages)*

---

## ✨ Features

- 📱 **Responsive Design** — works on mobile, tablet, and desktop
- 🎨 **Custom CSS Styling** — no CSS frameworks, fully hand-crafted
- ⚡ **Vanilla JavaScript** — smooth scrolling, interactive elements
- 🗂️ **Project Showcase** — cards for each project with links
- 📬 **Contact Section** — easy ways to get in touch
- 🎯 **Single Page** — fast load, no routing needed

---

## 🚀 Deployment (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/` (root)
4. Your portfolio is live at: `https://waseefullahh.github.io/portfolio`
